# QuantLight
Lightweight AI Quantitative Stock Selection Framework | 轻量AI量化选股工具

Simple, local-only, no network requests, no data crawling, safe open-source quant tool for A-share investors.

---

## 🎯 Key Features
- **Pure local runtime**: No network, no crawlers, fully compliant with data regulations
- **Multi-factor scoring model**: PE / PB / ROE / growth indicators integrated
- **Minimalist code**: Easy to read, modify and extend for learning
- **MIT licensed**: 100% free for personal and commercial use
- **Demo-ready**: Run immediately with simulated A-share data
